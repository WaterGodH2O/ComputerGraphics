Game Instructions

Controls:

  • W/A/S/D: Move
  • Mouse: Look around
  • Left Click: Shoot
  • Space: Jump
  • F: Toggle flashlight
  • 1/2/3/4: Switch weapons
  • H: Return to main menu

Debugging Controls:

  • P: Toggle camera view
  • T: Stop/Resume zombies

Objective:

Kill 30 zombies to win!

Tips:

  • Use grenades (weapon 3) for area damage
  • Zombies move automatically by default
  • Press H anytime to return to menu
